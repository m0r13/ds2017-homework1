Client -> <- Server

Connection:
-> hello (username)
<- hello_ack(username_available)

Get list of sessions:
-> get_sessions()
<- sessions((uuid, cur_num_players), ...)

Join existing session:
-> join_session(uuid)
<- session_joined(ok, uuid)

OR
Create new session:
-> create_session(num_players)
<- session_joined(ok, uuid)

When session is joined:
<- session_started()
<- sudoku_state(sudoku)
<- scores_state((player_name, points), ...)

When player wants to suggest a number:
-> suggest_number(position, number)
<- suggest_number_ack(ok)
<- sudoku_state(sudoku)
<- scores_state((player_name, points), ...)

When another player has suggested a number:
<- sudoku_state(sudoku)
<- scores_state((player_name, points), ...)

When a player wants to leave:
-> leave_session()

When all players have left except one / sudoku is finished:
<- game_over(winner)
