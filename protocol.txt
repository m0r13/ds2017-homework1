Client -> <- Server

Connection:
-> hello (username)
<- hello_ack(username_available)

Get list of sessions:
-> get_sessions()
<- sessions(sessions : [(uuid, cur_num_players), ...])

Join existing session:
-> join_session(uuid)
<- session_joined(ok, uuid)

OR
Create new session:
-> create_session(num_players)
<- session_joined(ok, uuid)

When session is joined:
<- session_started()
<- sudoku_state(sudoku)
<- scores_state(scores : [(player_name, points), ...])

Player suggest a number:
-> suggest_number(position, number)
<- suggest_number_ack(ok)
<- sudoku_state(sudoku)
<- scores_state(scores : [(player_name, points), ...])

Another player has suggested a number:
<- sudoku_state(sudoku)
<- scores_state(scores : [(player_name, points), ...])

Player wants to leave:
-> leave_session()

All players have left except one / sudoku is finished:
<- game_over(winner)
